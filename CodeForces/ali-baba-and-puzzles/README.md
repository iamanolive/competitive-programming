# D. Ali Baba and Puzzles

* Problem D from Assuit University Training Contest 1
* Link to problem: https://codeforces.com/group/MWSDmqGsZm/contest/326175/problem/D

## Solution Notes

if:
1. a + b - c == d || (d - a) / c == b || 
2. a - b + c == d || (d - a) / c == -b ||
3. (d + c) / a == b || (d - c) / a == b
then print YES
otherwise print NO
